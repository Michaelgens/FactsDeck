FACTS DECK — Short overview
=============================

What it is
----------
Facts Deck is a financial education site: articles, guides, and tools for personal
finance, investing, banking, and related topics.

Main content
------------
• Home — featured articles, latest posts, expert picks, trending, guides, browse by category
• Articles — /post — filter by type (featured, latest, expert picks, guides, trending), search, categories
• Individual posts — /post/[id]
• Tools — /tools/* — calculators and planners (see list below)
• Static pages — About, Contact, Privacy, Disclaimer, etc.

Content types (article listing)
-------------------------------
Featured, Latest, Expert Picks, Popular Guides, Trending Now

Browse-by-category labels (site UI)
-----------------------------------
Comparison, Guides, Decision Making, Economy, Markets, Wealth, Investing, Banking,
Personal Finance, Credit Cards, Real Estate, Retirement

Topics highlighted in navigation (examples)
-------------------------------------------
Personal Finance, Banking, Investing, Real Estate, Cryptocurrency, Retirement

Quick tools (names as shown on site)
------------------------------------
• Mortgage Calculator
• Investment Calculator
• Budget Planner
• Retirement Calculator
• Loan Calculator
• Credit Score Simulator

Newsletter
----------
Footer signup saves subscribers (Supabase) and can trigger a welcome email via the
separate Node backend when configured.

Admin
-----
